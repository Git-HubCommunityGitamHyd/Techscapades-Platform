### 4. Hint Viewing Restriction (Anti-Cheat)

**Context:** Prevent both team members from viewing the hint. Options:

#### Option A: Only One Can View
- [ ] Track which player viewed the hint
- [ ] If teammate tries to view, show message: "Your teammate has already viewed this hint"

#### Option B: Either Can View, Points Deducted
- [ ] Any team member can click "Show Hint"
- [ ] Points deducted from team immediately
- [ ] Both members then see the hint (synced)

**Recommended:** Option B (simpler to implement, same outcome)

**Implementation:**
- [ ] When hint is viewed, update team record
- [ ] Real-time sync to other team member's device
- [ ] Hint button disabled after first click
